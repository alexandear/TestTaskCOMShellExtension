Solution for this task should be a COM component that is ready for 64-bit. Use MS Visual Studio 2008/2010/2012 (it can be Express Edition).


Main task:


Write COM component, that can be registered in MS Windows 7/8 64-bit. It should extend system shell to provide:

  1. It should add menu item (example, "Calculate the Sum") in Windows Explorer?s context menu (right click on the list of files, folders should be ignored).

  2. This menu item should write information about selected files into the log file.

  3. Component should work on huge amout of selected files.

  4. If your implementation is not based on ATL you will have a good chance to get this job.


Advanced task:


You can increase your chance to get this job if you solve these issues:

  1. Each line in the log file should represents: short filename, size in human readable form, data of creation in human readable form.

  2. Component should only append lines into the log file.

  3. Lines should be sorted by file names in alphabetical order.

  4. In additional, it needs to calculate a per-byte sum (a flavour of checksum) in 4 bytes variable (DWORD) for each file.

  5. The calculation of checksum should be done using concurrency.

  6. Using thread pool is a desirable solution.

  7. Calculated information for each file should be logged as soon as possible, instead of at the end of processing of all files (sort order should be preserved).


Additional requirements:


  1. Source control systems (subversion, mercurial, git, etc.) are very useful for collaboration work. Use one of it to promote your solution (http://code.google.com, https://github.com, etc.).

  2. C-style code is not welcome.

  3. Use STL as much as possible.

  4. Use C++11 or Boost for concurrency.

  5. Implement your solution in cross-platform manner to be ready for use the business logic in Linux or Mac OS X application.

  6. Cover your solution by unit tests (example, Google Test).

  7. Test the business logic, not the COM specific code.

  8. Get ready to describe all functions that you will use to implement the solution.


Hint:


  1. To solve the main task you have to implement IShellExtInit and IContextMenu.

  2. Essential COM is a good book to learn COM technology.
